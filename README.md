# FleetTrack – Truck Document Reminder System

FleetTrack is a web-based application developed to manage truck documents and provide expiry reminders for important vehicle documents such as Insurance, PUC, Permit, RC Book, and Fitness Certificate.

The system helps transport owners and fleet managers track document expiry dates and avoid missing renewals.

---

## 🚛 Features

- Add and manage truck details
- Store document expiry dates
- Expiry alert system
- Dashboard for document tracking
- Update and delete records
- User-friendly interface
- Email notification support
